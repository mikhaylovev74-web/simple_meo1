# Book Tracker
**Автор:** Михев
**Описание:** Простое приложение для учёта прочитанных книг с фильтрацией и сохранением в JSON.
**Язык:** Python 3.x

## Инструкция по запуску

1. Установите Python 3.11.
2.Запустите приложение:
   ```
   python IA_book.py
   ```
4. Используйте интерфейс для добавления книг.
5. Данные сохраняются в файл `books.json`.
